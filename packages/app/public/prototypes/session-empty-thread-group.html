空会话 Thread — 群聊房间语义（重设计 v2）

上一版把空 thread 当成「我和本机 Agent 一对一」——在群聊模型里不成立。 会话创建后参与者可能是 你 + 多位成员 + 多位 Agent；空态应表达 「房间已开好，等第一个人开口」，而不是伪造某 Agent 的开场白。

设计约束：不假装已有消息；不绑定 MACMINI；与 @mention 路由规则一致 （ sole agent 自动响应 / 多人需 @ ）；chip 只预填 Composer；首条真实消息后整块消失。
